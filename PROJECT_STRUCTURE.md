# Project Structure

```
wildfire-spread-ai/
│
├── .env                          # Environment variables (local, not committed)
├── .env.example                  # Template for .env
├── requirements.txt              # Python dependencies
├── PROJECT_STRUCTURE.md
│
├── backend/                      # Flask application
│   ├── main.py                   # App entry point: ensure_db → create_all → seed_db → Flask
│   ├── config.py                 # Loads .env variables
│   │
│   ├── api/                      # API route blueprints
│   │   ├── auth.py               # POST /api/auth/register, login, verify ✅
│   │   ├── events.py             # GET  /api/events/ ✅
│   │   ├── chat.py               # POST /api/chat 🚧
│   │   ├── hotspots.py           # GET  /api/hotspots 🚧
│   │   └── risk_zones.py         # POST /api/risk-zones 🚧
│   │
│   ├── db/                       # Database layer (SQLAlchemy)
│   │   ├── connection.py         # db, get_db_uri(), ensure_db(), seed_db()
│   │   ├── models.py             # ORM: User, FireEvent
│   │   └── test_db.py            # DB connection test script
│   │
│   ├── agent/                    # AI agent (WIP)
│   │   ├── wildfire_agent.py
│   │   ├── tools.py
│   │   └── prompts.py
│   │
│   ├── ml/                       # ML inference
│   │   └── saved_models/
│   │       └── random_forest.pkl
│   │
│   └── utils/
│       └── auth_middleware.py    # JWT verification middleware
│
├── frontend/                     # Static frontend (served by Flask)
│   ├── index.htm                 # Main dashboard (requires auth)
│   ├── login.htm                 # Login / Register page
│   ├── css/
│   │   ├── main.css
│   │   ├── map.css
│   │   └── sidebar.css
│   └── js/
│       ├── auth.js               # Login, register, token management
│       ├── map.js                # Leaflet map setup
│       ├── wildfire.js           # Wildfire layer logic
│       ├── risk_zones.js         # Risk zone layer
│       ├── evacuation.js         # Evacuation route logic

├── data/

│   ├── scripts/                  # Data pipeline

│   │   │

│   │   │   ├── dl_landcover.py   # NRCan FBP fuel type TIFs
│   │   │   ├── dl_community.py   # StatCan Census Subdivision shapefiles
│   │   │   └── dl_population.py  # StatCan Dissemination Area shapefiles
│   │   │
│   │   ├── clipper/              # Clip static/remote data to event AOI
│   │   │   ├── clip_aoi.py       # AOI GeoJSON from fire_events.bbox
│   │   │   ├── clip_terrain.py   # MRDEM-30 COG (remote stream) → terrain/dem_dtm.tif
│   │   │   ├── clip_landcover.py # static landcover → landcover/fuel_type.tif
│   │   │   ├── clip_osm.py       # osmnx roads → osm/roads.gpkg
│   │   │   ├── clip_firms.py     # NASA FIRMS API → firms/YYYY-MM-DD.csv
│   │   │   ├── clip_community.py # static CSD → community/census_subdivisions.gpkg
│   │   │   ├── clip_population.py# static DA → population/dissemination_areas.gpkg
│   │   │   └── clip_perimeter.py # CWFIS hotspot buffer → perimeter/YYYY-MM-DD.geojson
│   │   │
│   │   └── doc/                  # API & data source reference docs
│   │       ├── firms_api_use.ipynb
│   │       ├── CanElevation-MRDEM-Usage-Guide.pdf
│   │       └── boundaries&population
│   │
│   ├── events/                   # Per-event data (auto-generated by pipeline)

│   │       ├── AOI/              # aoi.geojson
│   │       ├── terrain/          # dem_dtm.tif
│   │       ├── landcover/        # fuel_type.tif
│   │       ├── osm/              # roads.gpkg
│   │       ├── community/        # census_subdivisions.gpkg
│   │       ├── population/       # dissemination_areas.gpkg
│   │       ├── firms/            # YYYY-MM-DD.csv (one per day)
│   │       └── perimeter/        # YYYY-MM-DD.geojson (one per day)
│   │
│   └── static/                   # Shared national datasets (downloaded once)
│       ├── landcover/
│       │   ├── 2014/             # nat_fbpfuels_2014b.tif
│       │   └── 2024/             # FBP_fueltypes_Canada_100m_EPSG3978_20240527.tif

│       └── population/
│           ├── 2011/             # gda_000a11a_e.shp

│           └── 2021/             # lda_000b21a_e.shp
│
├── ml/                           # ML training artifacts
│   └── saved_models/

│       ├── fold1/

│           └── ...

├── docker/                       # Docker configuration

│   ├── init.sql                  # PostgreSQL schema + seed data (Docker only)
│   ├── DOCKER_SETUP.md
│   └── NON_DOCKER_SETUP.md
│
└── docs/
    └── api.yaml                  # OpenAPI 3.0 spec
```
